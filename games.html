When developing this project I used piskel, a pixel art format for the art of the game. I set out to make a gothic art style for my game and created all of the mechanics on a software called GDevelope. I learned how to do shortcuts for piskel and make the process a lot faster. For the coding part I created shortcuts and reused the same code for the next scenes. The big challenge was to create an inventor system for GDevelop because there was nothing for it and the inventory system in the software was not working so I created my own. By using CART’s habits of mind I persisted in this challenge determined to get an inventory to work in my game. I ended up successfully making my game with inventory.
